This project involves exploratory data analysis (EDA) of a dataset containing information about various countries.
The goal is to uncover interesting insights and patterns within the data,
focusing on demographic, economic, and political aspects.

Analysis Summary: 
So far, the following analyses have been performed:

Data Loading and Inspection: The Countries.csv file was loaded into a pandas DataFrame.
Basic inspection using df.head(), df.info(), df.describe(), and df.isna().sum()
was done to understand the data structure, types, and missing values.

Population Analysis: Identified the country with the highest population (India) and its capital (New Delhi).
Identified the country with the least population (Tuvalu) and its capital (Funafuti).
Determined the political leader of the second-highest populated country (China's Xi Jinping).

Democracy Score: Identified the top 5 countries with the highest democracy scores (Norway, Iceland, Sweden, New Zealand, Denmark).

Regional Analysis: Calculated the total number of distinct regions (22).
Counted countries in specific regions, such as Eastern Europe (10 countries).
Listed countries within the 'Eastern Europe' region.

Political Leadership: Counted the number of countries where the political leader's information is missing (7 countries).
Listed the countries with unknown political leaders.

Country Naming Convention: Identified countries with 'Republic' in their name.
African Region Analysis: Found the African country with the highest population (Nigeria).
